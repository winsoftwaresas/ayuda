## MÓDULO AMSI

### Parametrización visibilidad de solicitudes

{% include tag.html tag="added" %}

Se incorporó una nueva opción de parametrización que permite **restringir la visibilidad de solicitudes dentro del AMSI**. Con esta mejora, los usuarios podrán visualizar únicamente sus propias solicitudes de servicio, reforzando la privacidad y el control de la información dentro del sistema, para activar esta funcionalidad, el Administrador debe habilitar la opción **“Mostrar solo solicitudes del Usuario”** en la configuración del módulo. Esta parametrización garantiza que cada usuario acceda únicamente a sus propias solicitudes de servicio.

Con esta configuración activa, el usuario únicamente podrá visualizar las solicitudes de servicio que haya creado, evitando el acceso a solicitudes de otros usuarios.

## MODULO DE ORDENES

Se añadió en el módulo de Órdenes la nueva pestaña **Registro Metrológico**, que permite a los usuarios registrar los valores de **Error Máximo e Incertidumbre del Error** correspondientes a las **magnitudes** del activo asociado a la orden, siempre que esta tenga asignado un tipo de mantenimiento de metrología. Esta funcionalidad mejora la trazabilidad y facilita la gestión metrológica directamente desde la orden de trabajo.

{% set_image
  path:""
  css_class:""
  number:""
  filename:"1.8_10.png"
  alt:""
%}

Se creó el reporte **OTU-RMG – Resumen de Registros Metrológicos**, el cual consolida los datos registrados y los presenta **agrupados por Activo y por Magnitud**. Este reporte facilita el análisis metrológico, permitiendo visualizar de forma estructurada la información asociada a cada activo y sus respectivas magnitudes evaluadas.

{% set_image
  path:""
  css_class:""
  number:""
  filename:"1.8_11.png"
  alt:""
%}

Se creó un dashboard que muestra las Órdenes vencidas, organizándolas según su tiempo de vencimiento. Esta visualización permite identificar con mayor facilidad los retrasos y priorizar la gestión de las órdenes según su criticidad temporal.

{% set_image
  path:""
  css_class:""
  number:""
  filename:"1.8_12.png"
  alt:""
%}

Se creó el reporte **OTU-TIORP – Resumen de Órdenes de Trabajo con Tiempo**, el cual presenta un consolidado de las órdenes de trabajo incluyendo sus tiempos asociados. Este reporte permite visualizar de manera resumida los tiempos registrados en cada orden, facilitando el análisis operativo y la identificación de demoras o desviaciones.

{% set_image
  path:""
  css_class:""
  number:""
  filename:"1.8_13.png"
  alt:""
%}

Se creó el reporte **OTU-MCC – Resumen de Órdenes de Trabajo**, el cual lista las órdenes de trabajo e indica si cada una de ellas cuenta con **archivos multimedia y/o comentarios asociados**. Este reporte facilita la verificación del nivel de documentación de las órdenes y mejora el control sobre la información registrada en cada proceso.

{% set_image
  path:""
  css_class:""
  number:""
  filename:"1.8_14.png"
